## Data Cleaning & Processing
The original dataset contained missing, inconsistent, and unclear data. The following steps were performed:

### 1. Data Cleaning
- Trimmed text and aligned formats using Excel functions (e.g., `PROPER`)
- Removed duplicate records
- Filtered invalid transactions (e.g., missing transaction dates and payment method errors)
- Deleted unused columns while keeping the raw data intact

### 2. Data Recovery & Calculations
- Calculated missing data by using different formulas:
- Built an item unit price reference table to identify unknown items
- Recovered and standardized transaction information where possible

### 3. Analysis & Visualization
- Created heatmaps for **average daily sales by month**
- Analyzed **customer spending groups** and estimated potential revenue growth
- Built **two dashboards** to visualize sales trends and customer purchasing behavior
